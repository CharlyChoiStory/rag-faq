# 📖 챗봇 사용법

> Notion FAQ RAG 챗봇 사용 매뉴얼

---

## 🚀 실행 방법

### 1. 가상환경 활성화
```bash
cd /Users/charlychoi/Desktop/notion-faq-chatbot
source venv/bin/activate
```

### 2. Notion 데이터 인덱싱 (최초 1회 또는 FAQ 업데이트 시)
```bash
python src/notion_loader.py
```

### 3. 챗봇 실행
```bash
streamlit run src/app.py
```

### 4. 브라우저 접속
```
http://localhost:8501
```

---

## 💬 챗봇 사용 방법

1. 브라우저에서 챗봇 화면 열기
2. 하단 입력창에 질문 입력
3. Enter 또는 전송 버튼 클릭
4. AI 답변 및 출처 확인

### 잘 되는 질문 예시
```
✅ "환불 정책이 어떻게 되나요?"
✅ "회원가입은 어떻게 하나요?"
✅ "비밀번호를 잊어버렸을 때 어떻게 하나요?"
```

### 한계 상황
```
❌ FAQ에 없는 내용 → "해당 정보가 없습니다" 안내
❌ 매우 복잡한 상담 → 담당자 연결 안내 권장
```

---

## 🔄 FAQ 데이터 업데이트 방법

1. Notion에서 FAQ 내용 수정/추가
2. 터미널에서 재인덱싱 실행:
   ```bash
   python src/notion_loader.py --refresh
   ```
3. 챗봇 재시작

---

## ❓ 자주 묻는 질문 (챗봇 관련)

**Q. 챗봇이 엉뚱한 답변을 해요**
> Notion FAQ에 해당 내용이 없거나, 질문이 너무 모호할 수 있어요. 더 구체적으로 질문해보세요.

**Q. 느리게 응답해요**
> Claude API 응답 시간은 네트워크 상황에 따라 다를 수 있어요. 보통 3~10초 내 응답됩니다.

**Q. 오류가 발생해요**
> `.env` 파일의 API 키를 확인하고, `pip install -r requirements.txt`를 다시 실행해보세요.
