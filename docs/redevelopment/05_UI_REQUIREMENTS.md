# 05. UI 요구사항

## 기본 컨셉

카카오톡 스타일의 친숙한 챗봇 UI입니다.

## 상단 브랜딩

화면 중앙 상단에 큰 글씨로 다음 병원명을 표시합니다.

```text
AI미인 성형외과
```

## 사이드바

사이드바는 기술 설명을 최소화하고 사용자 행동에 집중합니다.

표시 항목:

```text
🏥 성형외과 FAQ
문서를 업로드하면 자동으로 파싱·온톨로지 태깅·벡터 DB 구축까지 실행됩니다.
FAQ 지식베이스 파일 업로드
예시 질문
```

제거한 항목:

```text
현재 단계
Phase 1 / Phase 2 / Phase 3
업무 온톨로지 설명
임베딩 모델 설명
LLM 설명
벡터 DB 설명
수동 구축 버튼
샘플 구축 버튼
대화 초기화 버튼
```

## 업로드 UX

파일을 업로드하면 별도 버튼 없이 자동 실행합니다.

```text
업로드
↓
자동 저장
↓
자동 파싱
↓
자동 온톨로지 태깅
↓
자동 벡터 DB 구축
```

## 답변 표시 원칙

사용자에게는 챗봇 답변만 표시합니다.

표시하지 않는 항목:

```text
참조 기준
정책유형
단계
기간
위험도
유사도
검색 결과 카드
출처 카드
```

## 색상/가독성 주의

Streamlit sidebar CSS에서 broad selector를 쓰면 file uploader 글자가 흰색으로 상속되어 안 보일 수 있습니다. 업로드 박스와 Browse files 버튼에는 별도 CSS로 검정 텍스트를 강제합니다.
